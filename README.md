# Habiticus
A habit tracker based on the principles covered in James Clear's book "Atomic Habits"

# What is included?
- Basic Habit Tracker
  - Based on the Martin System (+, -, o)
- Implementation Intention
  - I will [BEHAVIOR] at [TIME] in [LOCATION].
- Habit Stacking
- Habit Scorecard
- CBE Calculator
- Daily Habit Schedule
